# nasralutils
Coreutils for GenshiOS

### How to Build from source?
#### Build dependencies
``` nim
make - for all
nim (choosenim) - for nim project
zig - for zig project
ghc - for haskell project
g++ - for cpp project
```
#### Commands
``` bash
git clone https://github.com/NaGNU/nasralutils
cd nasralutils/
make
```
